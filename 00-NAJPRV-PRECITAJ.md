# Čo presne nahrať

Nahraj celý obsah tohto priečinka:

- `index.html`
- `app.js`
- `styles.css`
- `weatherBackground.css`
- `weatherBackground.js`
- `weatherAudio.js`
- `supabase-config.js`
- celý priečinok `assets`

# Prečo ti to predtým nemuselo prejsť

Najpravdepodobnejšia príčina predtým neboli obrázky paradajok a paprík, ale veľké súbory v:

- `assets/weather-video`

Tieto videá už boli odstránené ako reálne nepoužívané.

Aj tak je lepšie nahrávať web po menších dávkach, nie všetko naraz.

# Na čo si dať pozor

- nenahrávaj `.bak` súbory
- nenahrávaj `.md` a `.sql` dokumentáciu
- nenahrávaj `.edge-debug-profile`
- nahraj celý obsah tohto pripraveného priečinka

# Supabase

Supabase config je už zapísaný v:

- `supabase-config.js`

SQL update si už spustila zvlášť v Supabase, ten sa na web nenahráva.
